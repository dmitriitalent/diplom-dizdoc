https://backend.com/commandant/sendNotification POST

Комендант ограничен только своим общежитием, поэтому нужно фильтровать проживающих по общежитию.
``` json
{
	"title": "Принесите прививку от гриппа", 
	"text": "Дополнительный текст для уведомления, тут будут прописаны пояснения",
	residents: [ 
		562
	] // id выбранных проживающих
}
```































%% 
Если в массиве residents найдено несколько совпадающих ФИО, то бек должен вернуть status code 409 и 

``` json
[
	{
		"id": 654,
		"surname": "Иванов", 
		"name": "Иван", 
		"patronymic": "Иванович",
		"building": "2",
		"floor": "4",
		"room": "2413",
	},
	{
		"id": 4673,
		"surname": "Иванов", 
		"name": "Иван", 
		"patronymic": "Иванович",
		"building": "2",
		"floor": "1",
		"room": "2115",
	}
]
```

Затем комендант сможет выбрать конкретного проживающего. Чекбоксом проживающих добавляется в массив selectedResidents. %%
